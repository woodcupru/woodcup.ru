# WOODcup

Одностраничный архивный сайт для закрытого проекта WOODcup / BERЁZKA.

## Что в репозитории

- `index.html` — вся разметка страницы
- `style.css` — стили, адаптив и анимация морфинга центральной кляксы
- `pic/jcycle.png` — кольцо иллюстраций вокруг центрального блока
- `pic/favicon.ico` — favicon
- `CNAME` — домен GitHub Pages

## Архив

Папка `archive/` хранит:

- бэкап старой версии сайта
- референсы и макеты

`archive/` исключена из git-деплоя рабочей страницы и не используется текущим `index.html`.

## Публикация

Сайт рассчитан на GitHub Pages и собирается как статическая страница без сборщика и без JavaScript.

## Примечание

Текущая версия намеренно минимальная: в проде используются только один HTML-файл, один CSS-файл и два ассета из `pic/`.
